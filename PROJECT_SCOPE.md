# Product Goal

Build a multi-tenant SaaS platform inspired by AirClinic for clinics and salons.

## Core v1
- Auth
- Clinic onboarding
- Dashboard
- Doctors
- Services
- Patients
- Appointments
- Double-booking prevention
- Mock conversations inbox
- Future AI integration
- Future WhatsApp integration
- Future reminders and campaigns

## Target Customers
- Dental clinics
- Salons
- Small medical practices

## Channel Strategy
- Start with mock chat
- Later integrate WhatsApp Cloud API

## Build Principle
- Backend safety first
- AI never writes directly to DB without backend validation
- Keep architecture simple and scalable
